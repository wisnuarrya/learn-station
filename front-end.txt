Belajar front-end :
Front-end itu bagian membuat sebuah website tetapi dibagian luarnya seperti membuat design dari sebuah website,
ada aplikasi yang bernama Adobe XD, disana kita bisa membuat design terlebih dahulu sebelum kita benar-benar mengimplementasikannya ke realnya,
kita bisa men coba-coba design ataupun membuat design dari sebuah website yang kita inginkan dengan semau kita,
setelah kita membuat design website dari aplikasi Adobe XD,
kita bisa mengimplementasikan design tersebut ke dalam code pemrograman, dan membuatnya langsung secara real,
front-end sangat penting untuk tahap awal pembuatan website karena front-end adalah bagian awal yang dilihat dari user,
dan user bisa berinteraksi di tahapan front-end tersebut.

Bahasa Pemrograman front-end :
- HTML
- CSS
- JavaScript
